The Student Database Management System is a Python program designed to manage student records in a simple and efficient way. It provides a menu-driven interface that allows users to add, display, search, and delete student details easily. Each record includes the student’s USN, name, and marks in Python, Java, and C. The system automatically calculates the total percentage based on the entered marks. It also assigns a grade according to the percentage obtained. In addition, it checks and displays the pass or fail status for each subject individually. Student data is stored using dictionaries for quick access and organized management. The program uses functions to keep the code modular and easy to understand. Loops and conditional statements are used to control the flow of the application. Overall, this project demonstrates basic database operations and core Python programming concepts in a practical way.
